# 06 — VISUALIZE บน OBSIDIAN (ภาพที่จะเห็น)

## มุมที่ 1 — Reading view (เปิดไฟล์เครื่อง A)
```
┌──────────────────────────────────────────────────────────┐
│  machine-A.md                                  [Reading]  │
├──────────────────────────────────────────────────────────┤
│  machine: A   |  location: ปั๊มน้ำ                        │
│  last_repair: 2026-06-20   |  status: 🔴 เสีย             │
│                                                            │
│  # เครื่อง A                                              │
│                                                            │
│  ## ประวัติซ่อม                                           │
│  ┌──────────┬───────────┬────────┬──────────────┬───────┐ │
│  │ วันที่    │ อาการ      │ จุด     │ action        │ ที่มา  │ │
│  ├──────────┼───────────┼────────┼──────────────┼───────┤ │
│  │ 06-20    │ มอเตอร์ไหม้ │ ปั๊มน้ำ  │ เปลี่ยนมอเตอร์  │📄06-25│ │
│  │ 05-12    │ สั่นผิดปกติ  │ แบริ่ง  │ อัดจารบี       │📄05-12│ │
│  └──────────┴───────────┴────────┴──────────────┴───────┘ │
│                                                            │
│  🔗 Linked mentions (4)                                   │
│   • 2026-06-25 ประชุม — "เครื่อง A มอเตอร์ไหม้..."         │
│   • 2026-05-12 ประชุม — "เครื่อง A สั่น..."                │
└──────────────────────────────────────────────────────────┘
```

## มุมที่ 2 — Graph view (โยงเครื่องจักร ↔ ประชุม)
```
          ┌─────────────┐
          │ ประชุม 06-25 │
          └──────┬──────┘
        ┌────────┼────────┐
        ▼        ▼        ▼
   ┌────────┐┌────────┐┌────────┐
   │เครื่อง A ││เครื่อง B ││เครื่อง F │
   └───┬────┘└───┬────┘└────────┘
       │         │
   ┌───▼────┐┌───▼─────────┐
   │ประชุม   ││ประชุม 06-18  │     ● สีแดง = status เสีย
   │05-12   ││              │     ● สีเขียว = ปกติ
   └────────┘└─────────────┘     เส้น = ถูกพูดถึงในประชุม
```
คลิกโหนด "เครื่อง A" → กระโดดไปไฟล์ machine-A.md เห็นประวัติทันที

## มุมที่ 3 — Search ("เครื่อง A B F") → Dataview ขึ้นตาราง
```
┌──────────────────────────────────────────────────┐
│  🔍 dashboard.md                                  │
├──────────────────────────────────────────────────┤
│  ## เครื่องที่กำลังเสีย                            │
│  ┌──────────┬────────────┬────────┬─────────┐    │
│  │ เครื่อง   │ ซ่อมล่าสุด   │ จุด     │ status   │    │
│  ├──────────┼────────────┼────────┼─────────┤    │
│  │ A        │ 2026-06-20 │ ปั๊มน้ำ  │ 🔴 เสีย  │    │
│  │ F        │ 2026-06-22 │ หัวจ่าย  │ 🔴 เสีย  │    │
│  │ B        │ 2026-06-18 │ ไลน์ 2  │ 🟢 ปกติ  │    │
│  └──────────┴────────────┴────────┴─────────┘    │
└──────────────────────────────────────────────────┘
```

## ทำไม visualize แบบนี้ได้
- **frontmatter** (machine, status, last_repair) → Dataview query เป็นตาราง
- **`[[machine-A]]` link** → สร้างเส้นใน Graph view + Linked mentions อัตโนมัติ
- ค้น "เครื่อง A" → เปิดไฟล์เดียวเห็นประวัติรวม + ทุกประชุมที่พูดถึง (backlink)
