# 05 — EXAMPLE (ตัวอย่างจริงแต่ละขั้น)

ข้อมูลตัวอย่างทั้งหมดเป็น synthetic (ชื่อ/เครื่อง/วันที่สมมติ)

## ⑤ ผล WhisperX (เสียง → ข้อความ + คน + เวลา)
```
[00:21] SPEAKER_00: เพราะฉะนั้นไปเปิดสำโพง เดี๋ยวต้องเลือกเอา
[00:25] SPEAKER_01: อ๋อ นั่งได้นิดนึงนะ
[00:31] SPEAKER_00: เครื่อง A มอเตอร์ปั๊มน้ำไหม้เมื่อเช้า
[00:38] SPEAKER_01: ซ่อมล่าสุดไป 20 มิถุนา เปลี่ยนมอเตอร์ใหม่
```

## ⑥ หลัง map ชื่อ
```
[00:31] สมชาย: เครื่อง A มอเตอร์ปั๊มน้ำไหม้เมื่อเช้า
[00:38] สมหญิง: ซ่อมล่าสุดไป 20 มิถุนา เปลี่ยนมอเตอร์ใหม่
```

## ⑩ ไฟล์ .md ที่ได้

### vault/meetings/2026-06-25.md
```markdown
---
date: 2026-06-25
type: meeting
attendees: [สมชาย, สมหญิง]
machines: [A]
---
# ประชุม 2026-06-25

## สรุป
เครื่อง A มอเตอร์ปั๊มน้ำเสีย ซ่อมเปลี่ยนมอเตอร์

## เครื่องจักรที่พูดถึง
- [[machine-A]] — มอเตอร์ไหม้ (ปั๊มน้ำ) ซ่อม 2026-06-20

## Transcript
> [00:31] สมชาย: เครื่อง A มอเตอร์ปั๊มน้ำไหม้เมื่อเช้า

```

### vault/machines/machine-A.md (append ทุกครั้งที่ถูกพูดถึง)
```markdown
---
machine: A
location: ปั๊มน้ำ
last_repair: 2026-06-20
status: เสีย
---
# เครื่อง A

## ประวัติซ่อม
| วันที่ | อาการ | จุด | action | ที่มา |
|--------|-------|-----|--------|-------|
| 2026-06-20 | มอเตอร์ไหม้ | ปั๊มน้ำ | เปลี่ยนมอเตอร์ | [[2026-06-25]] |
| 2026-05-12 | สั่นผิดปกติ | แบริ่ง | อัดจารบี | [[2026-05-12]] |
```

### vault/dashboard.md (Dataview — เครื่องที่เสียอยู่)
````markdown
# Dashboard เครื่องจักร

## เครื่องที่กำลังเสีย
```dataview
TABLE last_repair AS "ซ่อมล่าสุด", location AS "จุด", status
FROM "machines"
WHERE status = "เสีย"
SORT last_repair DESC
```
````
